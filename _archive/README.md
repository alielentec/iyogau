# Archive

This folder preserves non-production material moved out of the live app root.

- `visual-evidence/` contains tracked screenshots and UI evidence from prior QA.
- `local-only/` is intentionally ignored by git for bulky local exports, downloaded references, and machine-local scratch material.

The archive is excluded from Vercel deployment by `.vercelignore`. Do not put runtime website, API, or source files here.
